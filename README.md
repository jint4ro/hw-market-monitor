# 🖥️ Hardware Market Monitor (hw-market-monitor)

Telegram-бот и асинхронный ETL-пайплайн для мониторинга цен на видеокарты. Проект собирает актуальные данные, сохраняет их в реляционную базу данных и предоставляет удобный интерфейс поиска через Telegram.

## 🚀 Функционал
* **Автоматизированный сбор данных:** Парсинг цен с помощью `Selenium` (с обходом защит через `undetected_chromedriver`).
* **Умный поиск (FSM):** Пошаговый диалог в Telegram для точного подбора видеокарты по:
  * Максимальному бюджету (свободный ввод).
  * Бренду (Inline-клавиатура).
  * Объему видеопамяти (Inline-клавиатура).
* **Надежное хранение:** Использование `PostgreSQL` с механизмом `UPSERT` для поддержания актуальности цен и защиты от дубликатов.

## 🛠 Стек технологий
* **Язык:** Python 3.12
* **Telegram Bot API:** Aiogram 3.x (FSM, Callback Queries)
* **База данных:** PostgreSQL (драйвер `psycopg2`)
* **Скрапинг:** Selenium, undetected-chromedriver
* **Инфраструктура:** Git, python-dotenv (безопасное хранение токенов)

## ⚙️ Структура проекта
* `parser.py` — скрипт сбора данных и обновления базы.
* `bot.py` — асинхронный Telegram-бот с интерфейсом.
* `.env` — файл конфигурации для токенов и доступов (не отслеживается Git).
